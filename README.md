# DNP Receipts v0.3

## Запуск

```bat
pip install -r requirements.txt
python receipt_generator.py
```

## Что добавлено в v0.3

- Кнопка **Создать шаблон с метками**.
- Поддержка двух типов шаблона:
  - обычный старый DOCX;
  - DOCX с метками `{{P}}`, `{{T1C}}`, `{{S1}}` и т.д.
- Галка **Сохранять PDF дополнительно к DOCX**. По умолчанию выключена.
- PDF создается через Microsoft Word, если он установлен, либо через LibreOffice.
- Вкладка **Отправка на почту**.
- Вкладка **Настройки почты**: SMTP сервер, порт, SSL/STARTTLS, отправитель, логин, пароль приложения.
- Последовательная отправка писем: одно письмо за другим.
- Статус по каждому участку: готово / отправка / отправлено / ошибка.
- Если email не найден в Excel, можно использовать тестовые адреса вида `1@mail.ru`.

## Как лучше хранить email в Excel

Самый простой вариант: рядом с блоком участка в любой ячейке строки участка или ближайших 7 строк ниже указать email.
Программа сама найдет первый email в блоке участка.

Например:

```text
A: участок 1 ... где-то справа: ivan@example.ru
ниже строки Т1/Т2/Т3
```
