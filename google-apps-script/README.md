# DNP Google Sheets UI v3.5.0

Файл `Code.gs` содержит актуальный интерфейс Google Apps Script для таблицы ДНП.

## Что добавлено

- меню называется `ДНП`;
- версия показывается только в `О программе`;
- служебные листы можно скрывать и открывать через меню;
- первичная настройка запрашивает пароль `123456`;
- можно использовать найденные папки и шаблон либо пересоздать структуру;
- год для PDF выбирается из листов с именами `2025`, `2026`, `2027` и так далее;
- добавлен пункт `Создать новый лист-год`;
- новый год копируется из предыдущего;
- сохраняются оформление, формулы, проверки данных, ширина столбцов и высота строк;
- обычные значения в столбцах `B` и далее очищаются;
- внешние HTML-файлы не нужны.

## Установка

1. Откройте Google Таблицу.
2. Перейдите в `Расширения → Apps Script`.
3. Сделайте резервную копию старого `Code.gs`.
4. Вставьте содержимое файла `Code.gs` из этой папки.
5. Сохраните проект и обновите страницу таблицы.

## Подключение существующего генератора PDF

В функции:

```javascript
startPdfGenerationFromDialog(year, month)
```

вызывается:

```javascript
generatePdfsForMonth(year, month)
```

Если рабочая функция генерации называется иначе, замените только это имя.

В конце `Code.gs` находятся заглушки:

```javascript
sendReceipts()
clearGeneratedPdfs()
```

Если эти функции уже определены в другом `.gs`-файле проекта, удалите заглушки, чтобы не было дублирования имён.
